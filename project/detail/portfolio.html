사용 기술 : HTML5, SCSS, Javascript

개인 포트폴리오 사이트입니다. AOS와 CSS keyframes를 이용한 애니메이션을 적극적으로 사용했습니다. SCSS로 작성하여 mixin 기능과 변수를 사용하고, header, footer, 퀵메뉴와 같이 여러 페이지에 중복되는 요소는 fetch 함수로 각각의 HTML 파일을 include하는 방식으로 작업하여 중복 코드 최소화와 유지보수를 고려했습니다. 자바스크립트 학습을 위해 자바스크립트 언어만으로 스크립트를 짰습니다.
