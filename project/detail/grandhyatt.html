그랜드하얏트 사이트를 리디자인했습니다. 호텔 특유의 이미지를 고려하여 현대적이면서 고급스러운 분위기를 나타내고자 남색 계열의 메인 색상을 선정했습니다. 타호텔 사이트와 비교하여 트렌드에 벗어나는 웹 디자인과 단조로운 기존 사이트를 개선하기 위해 다양한 레이아웃 디자인과 이벤트를 추가하는 것에 초점을 두었습니다.
메인 페이지는 flatpickr 플러그인과 select 태그를 이용해 체크인 기능을 구현하고, 슬라이더를 이용한 스페셜오퍼와 객실 섹션, 탭메뉴 기능을 이용한 다이닝 섹션, 클릭 이벤트를 이용한 부대시설 섹션, 아이콘을 추가한 리워즈 섹션으로 구성했습니다.
서브 페이지의 '주변 즐길거리' 페이지는 attr() 메서드로 data-name 값을 불러와 카테고리에 맞는 장소들을 보기 쉽게 필터링하도록 했습니다. '이용문의' 페이지는 input 태그를 이용하여 폼을 만들었습니다.
