사용 기술 : HTML5, CSS3, Javascript, jQuery

그랜드하얏트 사이트를 리디자인했습니다. 호텔 특유의 이미지를 고려하여 현대적이면서 고급스러운 분위기를 나타내고자 남색 계열의 메인 색상을 선정했습니다. 타호텔 사이트와 비교하여 트렌드에 벗어나는 웹 디자인과 단조로운 기존 사이트를 개선하기 위해 다양한 레이아웃 디자인과 이벤트를 추가하는 것에 초점을 두었습니다.
메인 페이지의 예약 섹션은 flatpickr 플러그인과 select 태그를 이용해 체크인 기능을 구현하고, slick을 이용한 메인 비주얼과 스페셜 오퍼, 객실 섹션, 탭메뉴로 구현한 다이닝 섹션, 클릭 이벤트를 이용한 부대시설 섹션, 아이콘을 추가한 리워즈 섹션으로 구성했습니다.
서브 페이지의 '주변 즐길거리' 페이지는 getAttribute 메서드로 data-name 값을 불러와 카테고리에 맞는 장소들을 보기 쉽게 필터링하도록 했습니다. '이용문의' 페이지는 input 태그를 이용하여 폼을 만들었습니다.
유지보수를 위해 header, footer, 모바일 메뉴와 같이 여러 페이지에 중복되는 요소는 fetch 함수로 각각의 HTML 파일을 include하는 방식으로 작업했습니다. 자바스크립트 공부를 하며 진행한 프로젝트로, 자바스크립트와 제이쿼리를 이용했습니다.
