메리플래닛 사이트를 리디자인했습니다. 기존 사이트가 정적이면서 콘텐츠가 부족하다고 생각했고, 이를 보완하는 쪽으로 리디자인 방향을 잡았습니다.
메인 비주얼은 영상을 사용하여 사용자의 이목을 집중시키면서 브랜드 컨셉을 각인시키고자 했습니다. 슬라이더를 활용한 제품, 인스타 섹션을 추가하여 메인 페이지에서 보여지는 콘텐츠를 늘리고, AOS와 hover로 시각적인 효과를 주었습니다. CSS 작업에서 변수와 rem 단위를 사용하여 유지보수와 반응형 작업을 고려했습니다.
